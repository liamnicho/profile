-A variable named "total" is declared and initialised to 0

While loops. A while loop can be emulated using a for loop with a condition statement. The loop will execute as long as the condition is true. Once the condition becomes false the loop will terminate.
-The variable "i" is declared 0 using shorthand
-A loop is created using the "for" with a keyword with the condition i < 5
-The value of "i" is printed using "fmt.Println"
-The value of "i" is incremented by 1 using "i++", the loop will terminate once the condition becomes false 		i := 0
for i < 5 {
fmt.Println(i)
i++
}

Why do we automate testing? You can fix bugs and make changes to code and know in a short period of time, that features which users depend on are still functioning.
Example GO test t.Run("empty string defaults to 'world'", func(t *testing.T) {
got := Liam("", "English")
want := "Hello, World"
assertCorrectMessage(t, got, want)
})
Explination of GO test -The t.Run is a function that shows this is a test
-This then calls the function "Liam"
-This assigns the result to the variable "got"
-Want is also a variable and has a value of "Hello, World"
-It then calls a function "assertCorrectMessage" with the "got" and "want" variables
